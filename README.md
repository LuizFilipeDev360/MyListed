# API de Catálogo de Mídias

## Descrição
API RESTful desenvolvida em .NET (C#) para gerenciamento de mídias, gêneros e relação entre usuários e mídias(listas). Inclui autenticação completa, CRUDs organizados e documentação via Swagger.

## Tecnologias
- ASP.NET Core
- Entity Framework Core
- ASP.NET Identity
- JWT
- MySQL
- Swagger

## Funcionalidades
- Cadastro e login de usuários
- Autenticação com JWT e refresh token
- CRUD de mídias
- CRUD de gêneros
- CRUD da relação usuário–mídia (listas, avaliações, status etc.)
- Regras de negócio e filtros aplicados aos endpoints
- Endpoints protegidos utilizando autorização baseada em Roles
